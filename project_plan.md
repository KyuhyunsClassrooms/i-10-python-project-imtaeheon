# 프로젝트 계획서

## 1. 프로젝트 주제

내가 만들 프로그램의 주제: 등차수열, 등비수열의 n항부터 m항까지의 합 계산기



---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.

내 설명: 등차수열 또는 등비수열의 n항부터 m항까지의 합을 알고 싶을때 (계산이 너무 복잡할때)

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

본 프로그램은 사용자가 선택한 수열의 종류(등차/등비)에 따라 **조건문(if-elif-else)**으로 실행 흐름을 제어하며, 지정된 구간의 합을 구하기 위해 1번째 항부터 끝 항(m)까지 차례대로 값을 계산하는 반복문(for) 알고리즘을 처리 규칙으로 사용합니다. 특히 반복문 내부에서 조건문(if n <= i <= m)을 활용해 사용자가 원하는 특정 시작 항과 끝 항 사이의 값만 변수에 누적하여 더해주는 구간 합산 방식으로 데이터를 처리하며, 계산이 완료된 총 6가지의 데이터 세트는 전체 기록을 관리하는 큰 리스트에 append하여 행과 열 구조를 갖춘 2차원 리스트 형태로 안전하게 축적하고 관리합니다."

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

사용자가 선택한 수열(등차/등비)의 첫째항, 공차/공비, 그리고 지정한 특정 구간(n번째 항부터 m번째 항까지)의 구간 합 계산 결과를 출력합니다. 또한, '이전 계산 기록 보기' 메뉴를 선택하면 그동안 계산했던 모든 내역을 표 형태로 누적 출력합니다.

## 4. 2차원 리스트 설계

내 프로그램에서 사용할 2차원 리스트의 이름:history



### 한 행의 의미
사용자가 한 번 수행한 **'하나의 수열 계산 기록 세트'**를 의미

### 각 열의 의미

0번 열의 의미는 계산한 수열의 종류이며, 예시로는 "등차" 또는 "등비"라는 문자열이 들어갑니다.

1번 열의 의미는 사용자가 입력한 수열의 첫째항(a)이며, 예시로는 2.0과 같은 실수형 데이터가 들어갑니다.

2번 열의 의미는 수열의 공차(d) 또는 공비(r)이며, 예시로는 3.0과 같은 실수형 데이터가 들어갑니다.

3번 열의 의미는 합산 계산을 시작할 항의 번호(n)이며, 예시로는 1과 같은 정수형 데이터가 들어갑니다.

4번 열의 의미는 합산 계산을 끝낼 항의 번호(m)이며, 예시로는 5와 같은 정수형 데이터가 들어갑니다.

5번 열의 의미는 최종적으로 계산된 구간 합 결과값(result)이며, 예시로는 40.0과 같은 실수형 데이터가 들어갑니다.
---

## 5. 함수 설계
1. 첫 번째 함수

함수 이름: calculate_arithmetic_range

역할: 첫째항, 공차, 시작항, 끝항을 입력받아 등차수열의 특정 구간의 합을 반복문으로 계산합니다.

입력값: a, d, n, m (첫째항, 공차, 시작할 항, 끝낼 항)

반환값 또는 출력: 반복문을 통해 계산된 최종 등차수열 구간 합(total)을 반환합니다.

2. 두 번째 함수

함수 이름: calculate_geometric_range

역할: 첫째항, 공비, 시작항, 끝항을 입력받아 등비수열의 특정 구간의 합을 반복문으로 계산합니다.

입력값: a, r, n, m (첫째항, 공비, 시작할 항, 끝낼 항)

반환값 또는 출력: 반복문을 통해 계산된 최종 등비수열 구간 합(total)을 반환합니다.

3. 세 번째 함수

함수 이름: show_history

역할: 2차원 리스트에 저장된 모든 계산 내역 행들을 순서대로 꺼내서 화면에 가독성 있게 출력합니다.

입력값: history_list (그동안의 데이터가 저장된 2차원 리스트)

반환값 또는 출력: 저장된 수열 종류, 첫째항, 공차/공비, 구간, 결과값을 문장 형태로 화면에 출력합니다.

## 6. 사용할 제어구조

### 조건문을 사용할 부분

while문 안에서 사용자가 선택한 메뉴 번호(k)가 1번인지, 2번인지, 3번인지, 4번인지 판별하여 각각 다른 기능(함수)을 실행할 때 조건문(if-elif-else)을 사용합니다.

수열 계산 함수 내부에서 반복문을 돌 때, 현재 항의 번호(i)가 사용자가 원하는 구간 안에 있는지(n <= i <= m) 필터링하여 합산할 때 조건문을 사용합니다.

### 반복문을 사용할 부분

예) 2차원 리스트의 모든 항목을 하나씩 검사한다.
내 계획:프로그램이 종료 번호(4번)를 받기 전까지 메뉴판을 띄우고 입력을 받으며 무한히 작동하도록 전체 흐름을 제어할 때 반복문(while True)을 사용합니다.

1번째 항부터 끝 항(m)까지 차례대로 나아가며 수열의 항들을 계산하고 누적할 때 반복문(for i in range(1, m + 1))을 사용합니다.

show_history 함수에서 2차원 리스트에 쌓인 이전 계산 기록 행들을 하나씩 꺼내서 화면에 출력할 때 반복문(for info in history_list)을 사용합니다.
